TP : Introduction à JavaScript et manipulation du DOM

Objectifs du TP :

  • Comprendre la syntaxe de base de JavaScript.
  • Manipuler le DOM pour interagir avec une page web.
  • Gérer les événements utilisateur via JavaScript.

Exercice 1 : Introduction à JavaScript

1. Créer un fichier index.html :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>TP JavaScript</title>
</head>
<body>
    <h1 id="titre">Bienvenue dans le TP JavaScript</h1>
    <button id="btnMessage">Afficher un message</button>
</body>
</html>

2. Créer un fichier script.js qui permet d'afficher dans la console "Hello World" quand on appuie sur le bouton

Exercice 2 : Manipulation du DOM

1. Ajouter un champ de saisie avant le bouton et un paragraphe après le bouton.

2. Modifier le comportement de bouton de l'exercice 1 pour qu'il affiche dans le paragraphe le texte entré dans le champs de saisie

Exercice 3 : Gestion des styles

1. Rajouter un bouton changer couleur juste après le bouton précédent

2. Créer la fonction qui permet de changer le background de message affiché en rouge quand on appuie sur le bouton.

Exercice 4 : Création et suppression d'éléments

1. Créer deux boutons ajouter ou supprimer sous le paragraphe.

2. Créer une liste vide sous les deux boutons

3. Ecrire les scripts necessaires pour manipuler la liste iet les relier aux boutons (ajout et suppression du dernier élément)

Exercice 5 : Formulaire

Créer un formulaire d'inscription à une salle de sport. Rajouter les champs que vous jugez nécéssaire. Ce formulaire contiendra un bouton Envoyer. En appuyant sur ce bouton tout les champs doivent être validés et afficher un message en cas d'erreur. Exemple d'erreur: un champ numéro de téléphone ne doit pas contenir des lettres, un mot de passe ne doit pas avoir une taille inférieur à 8 caractères, ...

En cas de validation de formulaire, afficher une alerte "formulaire envoyé avec succès"
